# aosp-devs.org February newsletter

Welcome to the fourteenth aosp-devs newsletter, keeping you up to date with
things happening in the AOSP developer community.

We are back to normal, 6 days too late without excuses.


## Security updates and AAOS release

The new three month security updates release cycle of AOSP is ticking. On 2nd
March, the security updates were released. See
[bulletins](https://source.android.com/docs/security/bulletin/2026/2026-03-01)
and [source code tag announcement](https://source.android.com/docs/security/bulletin/2026/2026-03-01).

Also the AAOS release was announced:
[Android Automotive 25Q4](https://source.android.com/docs/automotive/start/releases/aaos-25q4)

> This page describes features and enhancements in the Android Automotive OS (AAOS) 25Q4 release. AAOS 25Q4 is an official golden release. 25Q4 offers eight new features and addresses 180 issues.
>
> AAOS 25Q4 uses API level 36.
